# Semester Project 01

## Requirements

Before running this project, make sure you have installed:

- Raylib
- Visual Studio Code
- C/C++ Extension for VS Code
- MinGW-w64 (or another compatible C++ compiler)

## How to Run

1. Open Visual Studio Code.
2. Open the project folder.
3. Open `main.cpp`.
4. Build and run `main.cpp`.

Make sure Raylib is correctly installed and linked before compiling.

## Technologies Used

- C++
- Raylib
- Visual Studio Code
